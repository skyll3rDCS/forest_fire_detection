# Система автоматизированного мониторинга лесных пожаров на основе архитектуры Vision Transformer

## Аннотация

Настоящий проект представляет работу в области применения методов компьютерного зрения и глубокого обучения для решения задачи автоматического обнаружения лесных пожаров. В основе системы лежит предобученная модель [EdBianchi/vit-fire-detection](https://huggingface.co/EdBianchi/vit-fire-detection), адаптированная для классификации изображений. Система демонстрирует точность классификации 99.21% на тестовой выборке из 1900 изображений, что свидетельствует о высокой эффективности применения трансформерных архитектур в задачах экологического мониторинга.


### Актуальность проблемы

Лесные пожары представляют собой значительную угрозу для экосистем, биоразнообразия и экономики. Согласно данным FAO (Продовольственной и сельскохозяйственной организации ООН), ежегодно в мире регистрируется более 67 000 лесных пожаров, уничтожающих примерно 4,5 миллиона гектаров лесных массивов. Традиционные методы мониторинга, основанные на визуальном наблюдении и спутниковой съемке с ручным анализом, обладают рядом ограничений: высокие временные затраты, субъективность оценок, невозможность оперативного анализа больших объемов данных.

### Цель исследования

Разработка и валидация автоматизированной системы мониторинга, способной:
1. Осуществлять автоматическую классификацию изображений лесных массивов
2. Обеспечивать высокую точность детекции пожаров на ранних стадиях развития
3. Минимизировать количество ложных срабатываний и пропущенных событий

## Методология

### Архитектурное решение

В качестве базовой модели использована предобученная архитектура Vision Transformer (ViT-Base), адаптированная для задачи детекции пожаров. Модель была дообучена на специализированном датасете "Forest Fire Dataset", содержащем 14 000 аннотированных изображений.

#### Технические характеристики модели:
- **Архитектура**: Vision Transformer (ViT-B/16)
- **Количество параметров**: 86 миллионов
- **Размер входного тензора**: 224×224×3
- **Количество блоков трансформера**: 12
- **Размер скрытого слоя**: 768
- **Функция активации**: GELU

### Датсет 

["Forest Fire Dataset"](https://www.kaggle.com/datasets/alik05/forest-fire-dataset/data) с платформы Kaggle
